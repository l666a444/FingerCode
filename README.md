# FingerCode Biometric Security System

## Overview
This project implements a dual-authentication access control system using embedded hardware components.

## My Contribution
- Developed keypad-based authentication using Arduino Uno and C++
- Implemented fingerprint recognition and passcode verification modules
- Evaluated system behavior under hardware integration constraints

## System Features
- Passcode-based authentication
- Fingerprint recognition module
- Embedded control using Arduino

## Tools & Technologies
- Arduino Uno
- C++
- Fingerprint sensor
- Keypad module

## Results
Both authentication modules were successfully implemented and tested independently.

## Notes
Due to hardware integration limitations, fingerprint and passcode modules were not fully combined into a single system.
This is an academic project for demonstration purposes. Some components may be simplified.
